      Πρωτόκολλα

      Τα πρωτοκόλλα είναι ένα σύνολο από κανόνες που καθορίζουν τον τρόπο θα μπορούν να επικοινωνούν και να ανταλλάζουν πληροφορίες δυο συσκευές μεταξύ τους. Ορισμένα τα οποία ορίζει ένα πρωτόκολλο είναι το είδος της πληροφορίας, την μορφή και την σειρά των μνημάτων που θα ανταλλάξουν οι δυο συσκευές.


      Πρωτόκολλο Περιγραφή
      IP Internet Protocol
      Χρησιμοποιείται για τη μεταφορά πακέτων στο Internet
      IPv6 Internet Protocol version 6
      Η επόμενη γενιά του πρωτοκόλλου IP που χρησιμοποιεί μεγαλύτερη διευθυνσιοδότηση και άλλες βελτιώσεις
      ICMP Internet Control Message Protocol
      Χρησιμοποιείται για τη μεταφορά μηνυμάτων ελέγχου και σφαλμάτων στο Internet
      ARP Address Resolution Protocol
      Χρησιμοποιείται για τη εντοπισμό μιας διεύθυνσης MAC με βάση την διεύθυνση IP
      RARP Reverse Address Resolution Protocol
      Χρησιμοποιείται για την αντίστροφη μετατροπή μιας διεύθυνσης MAC σε αντίστοιχη διεύθυνση IP
      IPX Internetwork Packet Exchange
      Χρησιμοποιείται στα Δίκτυα Novell για τη μεταφορά πακέτων
      IGMP Internet Group Management Protocol
      Χρησιμοποιείται για τη διαχείριση group σε πρωτόκολλα multicast
      TCP Transmission Control Protocol
      Χρησιμοποιείται για την αξιόπιστη παράδοση πακέτων δεδομένων μεταξύ εφαρμογών
      UDP User Datagram Protocol
      Χρησιμοποιείται παράδοση πακέτων δεδομένων μεταξύ εφαρμογών, όταν μας νοιάζει η ταχύτητα παράδοσης αλλά δεν είναι αξιόπιστο.
      ICMPv6 Internet Control Message Protocol version 6
      Η έκδοση του πρωτοκόλλου ICMP που χρησιμοποιείται στο IPv6
      BGP Border Gateway Protocol
      Ένα πρωτόκολλο δρομολόγησης που χρησιμοποιείται για την ανταλλαγή των path μεταξύ διαφορετικών αυτόνομων συστημάτων (AS)
      FTP File Transfer Protocol
      Χρησιμοποιείται για τη μεταφορά αρχείων μεταξύ Server και Client στο Internet
      SFTP SSH File Transfer Protocol
      Είναι ένα ασφαλές πρωτόκολλο μεταφοράς αρχείων που χρησιμοποιείται για την ασφαλή μεταφορά αρχείων μέσω δικτύου.
      HTTP Hypertext Transfer Protocol
      Χρησιμοποιείται για τη μεταφορά website και data μεταξύ Server και Client
      HTTPS Hypertext Transfer Protocol Secure
      Μια ασφαλής έκδοση του HTTP που χρησιμοποιεί κρυπτογράφηση.
      SMTP Simple Mail Transfer Protocol
      Χρησιμοποιείται για την αποστολή και παράδοση μηνυμάτων στο ηλεκτρονικό ταχυδρομείο.
      DHCP Dynamic Host Configuration Protocol
      Χρησιμοποιείται για την αυτόματη ανάθεση διευθύνσεων IP σε συσκευές σε ένα δίκτυο
      SNMP Simple Network Management Protocol
      Χρησιμοποιείται για την διαχείριση και παρακολούθηση δικτύων, συσκευών και υπηρεσιών
      POP3 Post Office Protocol, version 3
      Χρησιμοποιείται για τη λήψη ηλεκτρονικών μηνυμάτων από έναν Server.
      IMAP Internet Message Access Protocol
      Ένα άλλο πρωτόκολλο για τη λήψη ηλεκτρονικών μηνυμάτων, με περισσότερες λειτουργίες από το POP3.
      DNS Χρησιμοποιείται για τη μετάφραση των ονομάτων τομέων (domain names) σε διευθύνσεις IP.
      DNSSEC DNS Security Extensions
      RTP Real-time Transport Protocol
      Χρησιμοποιείται για τη μετάδοση ήχου και βίντεο σε πραγματικό χρόνο.
      NTP Network Time Protocol
      Χρησιμοποιείται για το συγχρονισμό του χρόνου σε δίκτυα.
      IPSec Internet Protocol Security
      VoIP Voice over Internet Protocol
      RIP Routing Information Protocol
      Πρωτόκολλο δρομολόγησης που χρησιμοποιείται για την ανταλλαγή πληροφοριών δρομολόγησης μεταξύ router
      OSPF Open Shortest Path First
      Πρωτόκολλο δρομολόγησης που χρησιμοποιείται για τον υπολογισμό της βέλτιστης διαδρομής σε μεγάλα δίκτυα
      IPSec Internet Protocol Security
      Χρησιμοποιείται για την ασφαλή μετάδοση πακέτων δεδομένων μέσω του Internet
      VLAN Virtual Local Area Network
      PPP Point-to-Point Protocol
      Χρησιμοποιείται για τη σύνδεση δικτύων μέσω γραμμής τηλεφώνου ή άλλου μέσου επικοινωνίας
      MPLS Multiprotocol Label Switching
      Πρωτόκολλο δρομολόγησης που χρησιμοποιείται για τη δημιουργία VPN
      VRRP Virtual Router Redundancy Protocol
