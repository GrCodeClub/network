      Πρωτόκολλα

      Τα πρωτόκολλα στα δίκτυα αναφέρονται σε σύνολα κανόνων και συμβάσεων που καθορίζουν τον τρόπο με τον οποίο οι διάφορες συσκευές και υπολογιστές επικοινωνούν και ανταλλάσσουν πληροφορίες μέσα σε ένα δίκτυο. Τα πρωτόκολλα καθορίζουν τον τύπο των δεδομένων που μπορούν να ανταλλάσσονται, τη μορφή και τη σειρά των μηνυμάτων, καθώς και τους κανόνες για τον έλεγχο σφαλμάτων, την δρομολόγηση, την ασφάλεια και πολλά άλλα.


      Πρωτόκολλο Περιγραφή
      IP Internet Protocol
      Χρησιμοποιείται για τη μεταφορά πακέτων δεδομένων από ένα σημείο στο Διαδίκτυο σε άλλο
      IPv6 Internet Protocol version 6
      Η επόμενη γενιά του πρωτοκόλλου IP που χρησιμοποιεί μεγαλύτερη διευθυνσιοδότηση και άλλες βελτιώσεις
      ICMP Internet Control Message Protocol
      Χρησιμοποιείται για τη μεταφορά μηνυμάτων ελέγχου και σφαλμάτων στο Διαδίκτυο
      ARP Address Resolution Protocol
      Χρησιμοποιείται για τη μετατροπή μιας διεύθυνσης IP σε αντίστοιχη διεύθυνση MAC
      RARP Reverse Address Resolution Protocol
      Χρησιμοποιείται για την αντίστροφη μετατροπή μιας διεύθυνσης MAC σε αντίστοιχη διεύθυνση IP
      IPX Internetwork Packet Exchange
      Χρησιμοποιείται στα δίκτυα Novell για τη μεταφορά πακέτων δεδομένων
      IGMP Internet Group Management Protocol
      Χρησιμοποιείται για τη διαχείριση ομάδων σε πρωτόκολλα multicast
      TCP Transmission Control Protocol
      Χρησιμοποιείται για την αξιόπιστη παράδοση πακέτων δεδομένων μεταξύ εφαρμογών
      UDP User Datagram Protocol
      Χρησιμοποιείται για τη μη αξιόπιστη παράδοση πακέτων δεδομένων μεταξύ εφαρμογών
      ICMPv6 Internet Control Message Protocol version 6
      Η έκδοση του πρωτοκόλλου ICMP που χρησιμοποιείται στο IPv6
      OSPF Open Shortest Path First
      Ένα πρωτόκολλο δρομολόγησης που χρησιμοποιείται στα δίκτυα IP
      BGP Border Gateway Protocol
      Ένα πρωτόκολλο δρομολόγησης που χρησιμοποιείται για την ανταλλαγή διαδρομών μεταξύ διαφορετικών αυτόνομων συστημάτων
      FTP File Transfer Protocol
      Χρησιμοποιείται για τη μεταφορά αρχείων μεταξύ διακομιστών και πελατών στο Διαδίκτυο
      HTTP Hypertext Transfer Protocol
      Χρησιμοποιείται για τη μεταφορά ιστοσελίδων και δεδομένων μεταξύ διακομιστών και πελατών
      SMTP Simple Mail Transfer Protocol
      Χρησιμοποιείται για την αποστολή και παράδοση ηλεκτρονικού ταχυδρομείου
      DHCP Dynamic Host Configuration Protocol
      Χρησιμοποιείται για την αυτόματη ανάθεση διευθύνσεων IP σε συσκευές σε ένα δίκτυο
      SNMP Simple Network Management Protocol
      Χρησιμοποιείται για την διαχείριση και παρακολούθηση δικτύων, συσκευών και υπηρεσιών
      RIP Routing Information Protocol
      Ένα απλό πρωτόκολλο δρομολόγησης που χρησιμοποιείται για την ανταλλαγή πληροφοριών δρομολόγησης μεταξύ δρομολογητών
      OSPF Open Shortest Path First
      Ένα πιο προηγμένο πρωτόκολλο δρομολόγησης που χρησιμοποιείται για τον υπολογισμό της βέλτιστης διαδρομής σε μεγάλα δίκτυα
      IPSec Internet Protocol Security
      Χρησιμοποιείται για την ασφαλή μετάδοση πακέτων δεδομένων μέσω του Διαδικτύου
      VLAN Virtual Local Area Network
      Χρησιμοποιείται για τη δημιουργία εικονικών δικτύων μέσω λογικής απομόνωσης και ομαδοποίησης συσκευών
      PPP Point-to-Point Protocol
      Χρησιμοποιείται για τη σύνδεση δικτύων μέσω γραμμής τηλεφώνου ή άλλου μέσου επικοινωνίας
      MPLS Multiprotocol Label Switching
      Ένα πρωτόκολλο δρομολόγησης που χρησιμοποιείται για τη δημιουργία εικονικών ιδιωτικών δικτύων
      VRRP Virtual Router Redundancy Protocol
      Χρησιμοποιείται για τη δημιουργία ενός εικονικού δρομολογητή με σκοπό την αποφυγή διακοπής της λειτουργίας σε περίπτωση αποτυχίας ενός δρομολογητή
