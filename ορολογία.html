      Ορολογία

      Ένα δίκτυο αναφέρεται σε ένα σύνολο συνδεδεμένων συσκευών που μπορούν να επικοινωνούν μεταξύ τους.

      Ένας κόμβος αναφέρεται σε μια συσκευή ή ένα σημείο στο δίκτυο, όπως υπολογιστής ή router.

      Ένα Router είναι μια συσκευή που συνδέει διάφορα δίκτυα μεταξύ τους και μπορεί να αποφασίζει τη διαδρομή που πρέπει να ακολουθήσουν τα IP πακέτα.

      Ένα πρωτόκολλο αναφέρεται σε ένα σύνολο κανόνων και διαδικασιών που χρησιμοποιούνται για την επικοινωνία μεταξύ δικτύων και συσκευών.

      Το IP είναι ένα πρωτόκολλο διαδικτύου που χρησιμοποιείται για τη μεταφορά δεδομένων μεταξύ συσκευών σε ένα δίκτυο. Το IPv4 και το IPv6 είναι οι δύο κύριες εκδόσεις του πρωτοκόλλου IP.

      Η διεύθυνση MAC αναφέρεται είναι ένα μοναδικό αναγνωριστικό που ανατίθεται σε κάθε δικτυακή κάρτα. Χρησιμοποιείται για την αναγνώριση μιας συσκευής σε ένα LAN.

      Ένας πάροχος υπηρεσιών Διαδικτύου είναι μια εταιρεία ή οργανισμός που παρέχει πρόσβαση στο Διαδίκτυο στους χρήστες του. Ο ISP παρέχει συνήθως σύνδεση στο Διαδίκτυο μέσω καλωδίου, DSL, οπτικών ινών ή ασύρματων τεχνολογιών.

      Είναι η τεχνική που χρησιμοποιείται για να μετατρέψει μια διεύθυνση IP σε άλλη διεύθυνση IP κατά τη διέλευση από ένα δίκτυο σε ένα άλλο.

      Συσκευή ή σημείο πρόσβασης ασύρματης σύνδεσης.

      Εύρος ζώνης συχνοτήτων. Επίσης χρησιμοποείται για να δείξει τo εύρος των δεδομένων που μπορούν να μεταδοθούν σε ένα τηλεπικοινωνιακό κανάλι.

      Εκπομπή προς όλους τους κόμβους του δικτύου, συνοδεύοντας την πληροφορία με την αντίστοιχη broadcast διεύθυνση, ώστε να αποσταλεί σε όλους τους κόμβους.
