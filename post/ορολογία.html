      Ορολογία

      Όρος Περιγραφή
      Δίκτυο (Network) Ένα δίκτυο αναφέρεται σε ένα σύνολο συνδεδεμένων συσκευών που μπορούν να επικοινωνούν μεταξύ τους.
      Κόμβος (Node) Ένας κόμβος αναφέρεται σε μια συσκευή ή ένα σημείο στο δίκτυο, όπως υπολογιστής ή router.
      Δρομολογητής (Router) Ένα Router είναι μια συσκευή που συνδέει διάφορα δίκτυα μεταξύ τους και μπορεί να αποφασίζει τη διαδρομή που πρέπει να ακολουθήσουν τα IP πακέτα.
      Πρωτόκολλο (Protocol) Ένα πρωτόκολλο αναφέρεται σε ένα σύνολο κανόνων και διαδικασιών που χρησιμοποιούνται για την επικοινωνία μεταξύ δικτύων και συσκευών.
      IP Internet Protocol
      Το IP είναι ένα πρωτόκολλο διαδικτύου που χρησιμοποιείται για τη μεταφορά δεδομένων μεταξύ συσκευών σε ένα δίκτυο. Το IPv4 και το IPv6 είναι οι δύο κύριες εκδόσεις του πρωτοκόλλου IP.
      MAC Διεύθυνση Media Access Control
      Η διεύθυνση MAC αναφέρεται είναι ένα μοναδικό αναγνωριστικό που ανατίθεται σε κάθε δικτυακή κάρτα. Χρησιμοποιείται για την αναγνώριση μιας συσκευής σε ένα LAN.
      DHCP Dynamic Host Configuration Protocol
      Το DHCP είναι ένα πρωτόκολλο που χρησιμοποιείται για την αυτόματη διαμόρφωση των διευθύνσεων IP σε συσκευές σε ένα δίκτυο.
      DNS Domain Name System
      Το DNS είναι ένα σύστημα που αντιστοιχεί τα ονόματα (domain names) με τις διευθύνσεις IP.
      Firewall Ένας τείχος προστασίας είναι μια ασφαλής ζώνη μεταξύ διαφορετικών δικτύων, που επιτρέπει ή αποκλείει τη διέλευση των δεδομένων βάσει προκαθορισμένων κανόνων ασφαλείας.
      VPN Virtual Private Network
      Ένα εικονικό ιδιωτικό δίκτυο.
      Πρωτόκολλο TCP/IP Το TCP/IP αναφέρεται στο σύνολο των πρωτοκόλλων που χρησιμοποιούνται για τη μεταφορά δεδομένων στο Διαδίκτυο.
      VLAN Virtual Local Area Network
      ISP Internet Service Provider
      Ένας πάροχος υπηρεσιών Διαδικτύου είναι μια εταιρεία ή οργανισμός που παρέχει πρόσβαση στο Διαδίκτυο στους χρήστες του. Ο ISP παρέχει συνήθως σύνδεση στο Διαδίκτυο μέσω καλωδίου, DSL, οπτικών ινών ή ασύρματων τεχνολογιών.
      NAT Network Address Translation
      Είναι η τεχνική που χρησιμοποιείται για να μετατρέψει μια διεύθυνση IP σε άλλη διεύθυνση IP κατά τη διέλευση από ένα δίκτυο σε ένα άλλο.
