      Ορολογία

      Όρος Περιγραφή
      Δίκτυο (Network) Ένα δίκτυο αναφέρεται σε ένα σύνολο συνδεδεμένων συσκευών (όπως υπολογιστές, εκτυπωτές, δρομολογητές) που μπορούν να ανταλλάσσουν δεδομένα μεταξύ τους.
      Κόμβος (Node) Ένας κόμβος αναφέρεται σε μια συσκευή ή ένα σημείο στο δίκτυο, όπως υπολογιστής, δρομολογητής ή εκτυπωτής.
      Δρομολογητής (Router) Ένας δρομολογητής είναι μια συσκευή που συνδέει διάφορα δίκτυα μεταξύ τους και μπορεί να αποφασίζει τη διαδρομή που πρέπει να ακολουθήσουν τα πακέτα δεδομένων.
      Πρωτόκολλο (Protocol) Ένα πρωτόκολλο αναφέρεται σε ένα σύνολο κανόνων και διαδικασιών που χρησιμοποιούνται για την επικοινωνία μεταξύ δικτύων και συσκευών. Τα πρωτόκολλα καθορίζουν τον τρόπο με τον οποίο τα δεδομένα μεταδίδονται, υποδιαιρούνται, διαμορφώνονται κ.λπ.
      IP Internet Protocol
      Το IP είναι ένα πρωτόκολλο διαδικτύου που χρησιμοποιείται για τη μεταφορά δεδομένων μεταξύ συσκευών σε ένα δίκτυο. Το IPv4 και το IPv6 είναι οι δύο κύριες εκδόσεις του πρωτοκόλλου IP.
      MAC Διεύθυνση Media Access Control
      Η διεύθυνση MAC αναφέρεται σε ένα μοναδικό αναγνωριστικό που ανατίθεται σε κάθε δικτυακή κάρτα ενός δικτυακού προσαρμογέα. Χρησιμοποιείται για την αναγνώριση μιας συσκευής σε ένα τοπικό δίκτυο.
      DHCP Dynamic Host Configuration Protocol
      Το DHCP είναι ένα πρωτόκολλο που χρησιμοποιείται για την αυτόματη διαμόρφωση των διευθύνσεων IP σε συσκευές σε ένα δίκτυο.
      DNS Domain Name System
      Το DNS είναι ένα σύστημα που μετατρέπει τα ονόματα τομέων (domain names) σε αντίστοιχες διευθύνσεις IP. Χρησιμοποιείται για να αναζητήσει και να αντιστοιχίσει διευθύνσεις IP με βάση τα ονόματα τομέων που εισάγονται στον περιηγητή.
      Firewall Ένας τείχος προστασίας είναι μια ασφαλής ζώνη μεταξύ διαφορετικών δικτύων, που επιτρέπει ή αποκλείει τη διέλευση των δεδομένων βάσει προκαθορισμένων κανόνων ασφαλείας.
      VPN Virtual Private Network
      Ένα εικονικό ιδιωτικό δίκτυο είναι ένας τρόπος σύνδεσης ιδιωτικών δικτύων μέσω ενός δημόσιου δικτύου (όπως το Διαδίκτυο). Η σύνδεση επιτυγχάνεται μέσω κρυπτογράφησης για την ασφαλή μεταφορά των δεδομένων.
      Πρωτόκολλο TCP/IP Το TCP/IP αναφέρεται στο σύνολο των πρωτοκόλλων που χρησιμοποιούνται για τη μεταφορά δεδομένων στο Διαδίκτυο. Περιλαμβάνει το πρωτόκολλο IP, το πρωτόκολλο TCP για αξιόπιστη μετάδοση δεδομένων και άλλα πρωτόκολλα.
      VLAN Virtual Local Area Network
      Ένα VLAN αναφέρεται σε ένα λογικό δίκτυο που δημιουργείται εντός ενός φυσικού δικτύου LAN. Χρησιμοποιείται για τη δημιουργία διαχωρισμένων ομάδων συσκευών που μπορούν να επικοινωνούν αποκλειστικά μεταξύ τους.
      Φορητότητα (Mobility) Η φορητότητα αναφέρεται στην ικανότητα μιας συσκευής να μετακινείται από ένα δίκτυο σε άλλο χωρίς να χάνει τη σύνδεσή της. Οι τεχνολογίες φορητότητας συνήθως χρησιμοποιούνται σε ασύρματα δίκτυα.
      ISP Internet Service Provider
      Ένας πάροχος υπηρεσιών Διαδικτύου είναι μια εταιρεία ή οργανισμός που παρέχει πρόσβαση στο Διαδίκτυο στους χρήστες του. Ο ISP παρέχει συνήθως σύνδεση στο Διαδίκτυο μέσω καλωδίου, DSL, οπτικών ινών ή ασύρματων τεχνολογιών.
      NAT Network Address Translation
      Η μετάφραση διευθύνσεων δικτύου αναφέρεται στην τεχνική που χρησιμοποιείται για να μετατρέψει μια διεύθυνση IP σε άλλη διεύθυνση IP κατά τη διέλευση από ένα δίκτυο σε ένα άλλο. Αυτό επιτρέπει σε περιορισμένο αριθμό διευθύνσεων IP να χρησιμοποιούνται σε ένα μεγαλύτερο δίκτυο.
