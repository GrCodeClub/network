      Http Code Status

      Οι κωδικοί κατάστασης HTTP χωρίζονται σε πέντε κατηγορίες, ανάλογα με το πρώτο ψηφίο τους:

      • 1xx: Πληροφοριακές απαντήσεις (Informational)
      • 2xx: Επιτυχημένες απαντήσεις (Successful)
      • 3xx: Ανακατευθύνσεις (Redirection)
      • 4xx: Σφάλματα πελάτη (Client Errors)
      • 5xx: Σφάλματα διακομιστή (Server Errors)

      Κωδικός Περιγραφή
      200 OK (Επιτυχής αίτηση)
      Αυτός ο κωδικός υποδεικνύει ότι η αίτηση HTTP ολοκληρώθηκε με επιτυχία και ο Server επέστρεψε τα αναμενόμενα δεδομένα στον Client.
      201 Created
      Αυτός ο κωδικός υποδεικνύει ότι η αίτηση HTTP ήταν επιτυχής και ένα νέο αντικείμενο δημιουργήθηκε επιτυχώς στον Server.
      204 No Created
      Αυτός ο κωδικός υποδεικνύει ότι η αίτηση ολοκληρώθηκε με επιτυχία, αλλά ο διακομιστής δεν έχει επιστρέψει περιεχόμενο. Συνήθως χρησιμοποιείται όταν η αίτηση απλά εκτελεί μια ενέργεια, χωρίς να απαιτείται επιστροφή περιεχομένου.
      300 Multiple Choices
      Υποδεικνύει ότι ο διακομιστής έχει πολλαπλές επιλογές για την απόκριση στο αίτημα του Client και παρέχει μια λίστα με αυτές τις επιλογές. Αυτός ο κωδικός κατάστασης συνήθως συνοδεύεται από ένα σύνολο συνδέσμων (links) ή έναν πίνακα που περιέχει τις διαθέσιμες εναλλακτικές τοποθεσίες που μπορεί να ακολουθήσει ο Client.
      301 Moved Permanently (Μετακίνηση μόνιμη)
      Αυτός ο κωδικός υποδεικνύει ότι η πόρτα (URL) που ζητήθηκε έχει μετακινηθεί μόνιμα σε μια νέα τοποθεσία. Ο Client θα πρέπει να επαναδρομολογήσει το αίτημά του στη νέα τοποθεσία.
      302 Found
      Χρησιμοποιείται για να υποδείξει ότι η πόρτα που ζητήθηκε βρίσκεται προσωρινά σε μια διαφορετική τοποθεσία. Ο Client πρέπει να ακολουθήσει τη νέα τοποθεσία για να ολοκληρώσει το αίτημά του.
      Ο κωδικός 302 είναι συχνά χρησιμοποιούμενος για ανακατεύθυνση σελίδων μετά από ένα POST αίτημα.
      303 See Other
      Χρησιμοποιείται για να υποδείξει στον Client ότι η αίτησή του ολοκληρώθηκε με επιτυχία και ότι η απάντηση βρίσκεται σε μια διαφορετική τοποθεσία.
      Συνήθως, αυτό συμβαίνει όταν ένα POST αίτημα ολοκληρώνεται με επιτυχία και ο Server παρέχει μια απάντηση που προτρέπει τον Client να ακολουθήσει μια νέα τοποθεσία για περαιτέρω ενέργειες.
      304 Not Modified
      Χρησιμοποιείται για να υποδείξει στον Client ότι η πόρτα που ζητήθηκε δεν έχει αλλάξει από την τελευταία φορά που την ανέκτησε.
      305 Use Proxy
      Δεν χρησιμοποιείται συχνά και έχει αντικατασταθεί από τον κωδικό 307 "Temporary Redirect" στο πρότυπο HTTP/1.1.
      Ο κωδικός 305 προτρέπει τον πελάτη να χρησιμοποιήσει έναν προκαθορισμένο διακομιστή proxy για να ανακτήσει την αιτούμενη πόρτα.
      307 Temporary Redirect
      Χρησιμοποιείται για να υποδείξει ότι η αίτηση του Client πρέπει να ανακατευθυνθεί προσωρινά σε μια διαφορετική τοποθεσία.
      308 Permanent Redirect
      Χρησιμοποιείται για να υποδείξει ότι η αίτηση του Client πρέπει να ανακατευθυνθεί μόνιμα σε μια διαφορετική τοποθεσία.
      400 Bad Request (Κακή αίτηση)
      Yποδεικνύει ότι ο Server δεν μπορεί να επεξεργαστεί το αίτημα του πελάτη λόγω εσφαλμένης σύνταξης, ακατανόητων παραμέτρων ή άλλων σφαλμάτων που αφορούν το αίτημα.
      404 Not Found (Δεν βρέθηκε)
      Αυτός ο κωδικός υποδεικνύει ότι η πόρτα (URL) που ζητήθηκε δεν μπορεί να βρεθεί στον Server. Συνήθως αυτό συμβαίνει όταν ο πόρος έχει διαγραφεί ή δεν υπάρχει.
      500 Internal Server Error (Εσωτερικό σφάλμα διακομιστή)
      Αυτός ο κωδικός υποδεικνύει ότι συνέβη ένα εσωτερικό σφάλμα στον Server, που δεν επέτρεψε να ολοκληρωθεί η αίτηση του Client.
      501 Not Implemented
      Υποδεικνύει ότι ο Server δεν υποστηρίζει τη λειτουργία που απαιτείται για να εκτελεστεί η αίτηση του Client. Αυτό σημαίνει ότι ο Server δεν έχει υλοποιήσει ή δεν αναγνωρίζει τη μέθοδο αίτησης που χρησιμοποιείται ή τον τύπο του περιεχομένου που απαιτείται.
      502 Bad Gateway
      Yποδεικνύει ότι ο Server λειτουργεί ως μεσολαβητής (gateway) ή αντιμετωπίζει μια άλλη αίτηση, και έλαβε μια άκυρη απάντηση από τον προορισμό της.
      503 Service Unavailable
      Αυτός ο κωδικός υποδεικνύει ότι ο Server δεν είναι προσωρινά διαθέσιμος για να εξυπηρετήσει το αίτημα του Client. Αυτό μπορεί να συμβεί, για παράδειγμα, όταν ο Server είναι υπερφορτωμένος ή υπό συντήρηση.
      504 Gateway Timeout
      Υποδεικνύει ότι ο Server λειτουργεί ως μεσολαβητής (gateway) ή προξενητής και δεν λαμβάνει μια έγκυρη απάντηση εγκαίρως από τον προορισμό της.
      505 HTTP Version Not Supported
      Υποδεικνύει ότι ο Server δεν υποστηρίζει την έκδοση του πρωτοκόλλου HTTP που χρησιμοποιείται στο αίτημα του Client.
